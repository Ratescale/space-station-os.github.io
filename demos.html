Demos

Explore practical demonstrations of Space Station OS in action. These demos showcase various features and capabilities in simulated environments.

Demo 1a - Nauka Incident

This demo recreates the ISS Nauka incident for fault analysis. Follow these steps:

  1. Run the following commands in separate terminals after sourcing your ROS 2 environment:
    • ros2 run space_station_gnc demo1a_nauka_incident_estimate - Terminal 1
    • ros2 launch space_station_gnc launch_gnc.py - Terminal 2
    • ros2 run rviz2 rviz2 - Terminal 3 (Launches RVIZ)
  2. Observe the simulation and verify the response process.
