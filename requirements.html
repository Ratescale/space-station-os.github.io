Requirements

To set up Space Station OS:

  • Ubuntu 22.04
  • ROS 2 Humble (desktop install)

After cloning this repository, source the ROS2 setup in your terminal:

source /opt/ros/humble/setup.bash

Then, build the workspace from the top level of the repository:

colcon build

Afterward, each terminal must also source the install:

source install/setup.bash
